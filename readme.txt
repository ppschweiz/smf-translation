PPSGravatar (0.1.1) is for the Non-Tracking Gravatar-Functionality in SMF

PPSGravatar downloads the users Gravatar from the gravatar.com server and stores the image on the Server. So gravatar.com sees only the servers IP-Address and cannot track all users who are visiting the Forum.
